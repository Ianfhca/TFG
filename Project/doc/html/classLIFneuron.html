Detailed Description

Implements a Leaky Integrate-and-Fire neuron with STDP and WTA learning mechanisms.

This class represents an individual neuron in a Spiking Neural Network using the LIF model. It supports synaptic plasticity via STDP, winner-take-all inhibition, and saving/loading synaptic weights.
